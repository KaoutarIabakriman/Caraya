# Caraya - Premium Car Rental

A luxury car rental web application built with a Flask/MongoDB backend and a React/Next.js frontend.

## Features

- Elegant and responsive UI with premium design
- Advanced car search and filtering
- Car details with image carousel
- Booking system
- Manager and Admin portals
- Animated components with Framer Motion
- Fully responsive design for all devices

## Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

### Backend
- Flask
- MongoDB
- Python

## Getting Started

### Prerequisites
- Node.js
- Python 3.x
- MongoDB

### Installation

#### Backend
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python run.py
```

#### Frontend
```bash
cd frontend
npm install
npm run dev
```
